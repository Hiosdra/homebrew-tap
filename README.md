# Hiosdra Homebrew Tap

This tap distributes prebuilt Hiosdra command-line tools.

## Install codex-usage-guard

```sh
brew tap Hiosdra/tap
brew install --cask codex-usage-guard
```

The cask supports macOS and Linux on arm64 and x64. It installs the
`codex-usage-guard` executable; the Codex hook remains unchanged until you run
`codex-usage-guard install-hook`.

Project repository: <https://github.com/Hiosdra/codex-usage-guard>
